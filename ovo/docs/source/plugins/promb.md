# OVO Promb Plugin

[Promb](https://github.com/MSDLLCpapers/promb) is a Python library for computing protein humanness 
(or mouse-ness, dog-ness, cow-ness, ...) based on average number of mutations to nearest peptide
in a reference proteome (Human SwissProt, human antibody repertoires in OAS, or a custom reference DB).

## Install OVO Promb Plugin

Install the plugin into your OVO environment:

```bash
pip install https://github.com/MSDLLCpapers/ovo-promb
```
