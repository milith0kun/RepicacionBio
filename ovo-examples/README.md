# OVO Examples

Usage and configuration examples for the OVO de novo design ecosystem.

- [jupyter_notebooks_example](./jupyter_notebooks_example) - Notebooks for motif scaffolding and binder design submission from Jupyter
- [hpc_pbs_config_example](./hpc_pbs_config_example) - OVO configuration for HPC PBS scheduler
- [aws_healthomics_example](./aws_healthomics_example) - Terraform scripts for deployment of OVO workflows to AWS HealthOmics
